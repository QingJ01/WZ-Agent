# 安全与隐私

---

## 禁止提交的内容

| 类别 | 示例 |
|------|------|
| 设备信息 | ADB 序列号、本机用户名、绝对路径 |
| 游戏数据 | 账号、昵称、截图、聊天内容 |
| 运行产物 | 日志、训练样本、私有模型权重 |
| 本地配置 | `.env` 文件 |

`.gitignore` 已默认排除常见本地产物，但发布前仍需人工检查。

---

## 设备安全

本项目通过 ADB 向设备发送触摸、滑动和按键指令。使用前请确认：

- 连接的是你自己的或有权操作的设备
- GUI 中「AI 自动操作」状态符合预期
- 不需要操作时保持 `WZRY_AI_CONTROL_ENABLED=0`

---

## 模型安全

PyTorch `.pt` 文件使用 pickle 反序列化，可能包含恶意代码。**只使用自己训练或可信来源的模型**，不要加载来源不明的权重文件。

---

## 漏洞反馈

如果在公开仓库中发现泄露的私有数据、可利用的危险行为或依赖安全问题，请先**私下联系**仓库维护者，不要在 issue 中公开敏感细节。
